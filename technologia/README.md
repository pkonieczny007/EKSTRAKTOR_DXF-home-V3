W jaki sposób określamy Gięcie?


Każdy element musi mieć określoną technologię - marszrutę wykonania
G -gięcie
GS - gięcie+spawanie
GSO - gięcie+spawanie+obróbka (np. gwinty, fazowanie)
S - spawanie
SO - spawanie+obróbka
O - obróbka (np. gwinty, fazowanie)
puste - czasami są elementy, które tylko wypalamy i wysyłamy lub montujemy śrubami
W - walcowanie
WS - walcowanie+spawanie
WSO - walcowanie+spawanie+obróbka
